What is vh?

When you use vh, the number you set for the height of the box will be displayed relative to the entire screen.

What is vw?

This is the same as the discription above, but applies to the width of the page.

What is border box?

When we use this tag, we want to automatically set the size of all the boxes with the details of the padding and margin that we want to be placed inside the parent box without overflowing.
